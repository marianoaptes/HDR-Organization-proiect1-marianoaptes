Limbajul HTML>



HTML(HyperText Markup Language) este un limbaj de marcare utilizat pentru a crea pagini web.

Un exemplu de cod html il gasiti in imaginea urmatoare care se afla la adresa http://upload.wikimedia.org/wikipedia/commons/e/e4/HTML_logo.png.

Imaginea are atributul height de valoare 147.



Tag-urile html cele mai utilizate in realizarea tabelelor sunt:

table defineste un tabel
tr un rand intr-un tabel
td o celula intr-un tabel


Observatie: Pentru a pune un contur unui tabel se poate, dar nu se recomanda) sa se puna atributul border ="1".
La validare va da eroarea: The border attribute on the table element is obsolete.

Formularele se pot crea cu ajutorul tag-ului form care poate contine tag-urile: input, textarea, label,select, option, botton.
Pentru mai multe detalii consultati cursul sau tutorialul HTML Form and Inputs de la adresa http://www.w3schools.com/html/html_forms.asp


Un exemplu de formular este urmatorul:
Nume:
Prenume:
Parola:
Culoarea preferata:
Descriere:
Sunt de acord cu termenii si conditiile



Pentru a crea liste se pot folosi tagurile:
  1. ul - lista neordonata
  2. ol - lista ordonata
  3. li - item din lista
  4. dl - lista cu descrieri
  5. dt - termen dintr-o lista cu descrieri
  6. dd - descrierea dintr-o lista de descrieri
Pentru a insera un document HTML in altul se poate folosi tag-ul <iframe>.
Astfel, exemplul de mai jos inseareaza continutul de la adresa https://html.spec.whatwg.org/multipage/.
